# data2/v1/dict

A拼音 1.x 内置词典数据 (30 万词).

+ `count.py`  输出一些统计信息.

+ `extract.py`  从 A拼音 1.x 数据库 (`core_data.db`) 中提取此部分数据的代码.

+ `d_1.txt` `d_2.txt` 等:
  具体词典数据, 每个文件一万行.

+ `symbol2.txt`  中文标点数据


## `d_1.txt` 文件格式

```
# 注释: A拼音 1.x 内置词典数据 (来源不清)
# 格式: 计数 词

10000768 自己
8271971 什么
7657341 没有
7578341 一个
```
